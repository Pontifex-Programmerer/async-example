Synchronized vs Asynchronized functions

Sync - when pressing this button, you are starting a synchronized function call.
synchronized functions will block all other activity on a web page, such as GUI updates or other button presses, making your web-page unresponsive. It is ok to use synchronized functions for most cases, but you should avoid using them if they take a long time to complete.

Async - when pressing this button, you are starting an asynchronous function call.
Asynchronous function calls are not blocking any other processes on your web page, and can be aborted if needed. Any operation that takes an unpredictable amount of time to complete shall always be asynchronous.

When async processes are running, you are able to interact with web-pages and programs normally.
